# Newsletter Platform - Project Guide

## About the Learner

This project is being built as a learning exercise. The developer is new to JavaScript, TypeScript, and Next.js. All code changes should be preceded by clear explanations of the concepts involved.

## Rules for Claude

1. **Explain before implementing.** Before writing or modifying any code, explain the relevant concepts (language features, framework patterns, libraries) in plain terms. Do not assume prior JavaScript/TypeScript knowledge.
2. **Introduce concepts incrementally.** Don't overwhelm with too many new ideas at once. Build on what has already been covered.
3. **Comment non-obvious code.** Add inline comments to help the learner understand what each section does and why.
4. **Highlight TypeScript-specific features.** When using types, interfaces, generics, or other TS features, call them out and explain how they differ from plain JavaScript.
5. **Explain framework conventions.** Next.js has many conventions (file-based routing, server/client components, etc.). Explain these when they first appear.

## Learning Goals

### JavaScript, TypeScript & Next.js
- Understand JavaScript fundamentals (variables, functions, async/await, modules)
- Learn TypeScript's type system and how it adds safety on top of JavaScript
- Understand React component model (JSX, props, state, hooks)
- Learn Next.js App Router patterns (layouts, pages, server components, API routes)
- Build a working blog with Markdown/MDX support
- Add newsletter subscription and email delivery
- Deploy the finished project

### Claude Code & Advanced Features
- Learn Claude Code CLI basics (commands, slash commands, permissions, keyboard shortcuts)
- Understand and use hooks (pre/post tool-call shell commands for automation)
- Work with MCP servers (connecting external tools and data sources)
- Use subagents and parallel agent workflows for complex tasks
- Learn worktree isolation for safe experimental changes
- Use CLAUDE.md files effectively (project-level, folder-level, user-level)
- Master the memory system for persistent context across sessions
- Understand permission modes and how to configure them
- Use /loop for recurring and self-paced tasks
- Learn to schedule remote agents with cron triggers
