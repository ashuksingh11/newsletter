Next.js is a React framework that makes building web applications simple and fast.

## Why Next.js?

Traditional React apps run entirely in the browser. Next.js adds **server-side rendering**, file-based routing, and built-in optimizations so your site loads faster and ranks better in search engines.

Here's what you get out of the box:

- **File-based routing** — create a file, get a URL
- **Server Components** — render on the server for faster page loads
- **Built-in API routes** — write backend logic alongside your frontend
- **TypeScript support** — type safety from day one
